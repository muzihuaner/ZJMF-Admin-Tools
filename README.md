# ZJMF-Admin-Tools
## 魔方财务管理员重置密码工具

管理员帐号被禁用怎么办？
出现管理员帐号被禁用，一般是因为连续3次错误输入登录密码

解决办法有两种：

1、等6小时，禁用自动解除；

2、使用工具临时解除：

**下载tools.php文件**

1. 下载tools.php，放至网站的 /public路径下
2. 访问 域名/tools.php ，输入登录密码(初始密码：000000)，访问重置密码工具
3. 选择需要修改的管理员用户
4. 输入重置后的密码，点击提交



**特别注意：为临时工具箱，密码设置完成后，一定要删除此文件，否则会有严重的安全隐患**
